Dummy Cog

Dummy Cog is a type of Dummy Cog.
Dummy Cog is used as a placeholder for this infobox template page. Its primary use is to help make sure everything looks nice and works for real cogs! The dummy cog can only be found on this page. Theoretically, the dummy cog can be anywhere from level 1 to 50, however it is not real so this is actually impossible.
